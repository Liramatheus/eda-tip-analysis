# eda-tip-analysis
Tips Analysis — Exploratory Data Analysis (EDA)

Este projeto realiza uma análise exploratória de dados (EDA) utilizando Python para investigar padrões em um conjunto de dados sobre gorjetas.
O objetivo é compreender como variáveis como valor consumido, dia da semana, gênero e forma de atendimento influenciam o valor das gorjetas.

# Objetivos do Projeto

- Explorar o dataset e entender sua estrutura

- Calcular estatísticas descritivas

- Criar visualizações para identificar padrões

- Gerar insights acionáveis e úteis para tomada de decisão

- Praticar técnicas fundamentais de análise exploratória

# Etapas do Projeto

- Importação e análise inicial do dataset

- Tratamento e limpeza dos dados

- Análise estatística descritiva

- Criação de gráficos (seaborn/matplotlib)

- Interpretação dos principais insights

# Tecnologias Utilizadas

- Python

- Pandas

- NumPy

- Matplotlib

- Seaborn

- Jupyter Notebook
